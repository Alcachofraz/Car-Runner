Erases all data from sector 'startSector' to 'endSector'.

Parameters
startSector-> First sector.
endSector-> Last sector.
Returns
CMD_SUCCESS: if sector was successfully erased.
INVALID_SECTOR: if sector specified is not valid.
Note
To erase a single sector, the 2 parameters should hold the same value.

Parameters

srcAddr-> Second address of data.
size-> Byte count to be compared. Should be a multiple of 4.
Returns
CMD_SUCCESS: if data coincides.
COMPARE_ERROR: if data does not coincide.
INVALID_SECTOR: if sector specified is not valid.
COUNT_ERROR: if byte count is not a multiple of 4.
ADDR_ERROR: if address is not on word boundary.
ADDR_NOT_MAPPED: if address is not mapped.

◆ FLASH_WriteData()

unsigned int FLASH_WriteData ( unsigned int  sector,
void *  dstAddr,
void *  srcAddr,
unsigned int  size 
)

Write data in 'srcAddr' into 'dstAddr'.

Parameters
sector-> Sector containing address 'dstAddr'.
dstAddr-> Destination address.
srcAddr-> Source address, where data to be written is.
size-> Number of bytes to be written. Should be 256, 512, 1024 or 4096.
Returns
CMD_SUCCESS: if data was successfully written.
INVALID_SECTOR: if sector specified is not valid.
SRC_ADDR_ERROR: if source address is not a word boundary.
DST_ADDR_ERROR: if destination address is not on correct boundary.
SRC_ADDR_NOT_MAPPED: Source address is not mapped.
DST_ADDR_NOT_MAPPED: Destination address is not mapped.
COUNT_ERROR: Byte count is not 256, 512, 1024 or 4096.
Note
Address 'dstAddr' should be contained in sector 'sector'.
