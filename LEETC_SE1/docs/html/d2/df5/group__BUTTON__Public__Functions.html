uint32_t BUTTON_GetButtonsEvents ( uint32_t *  current)

Gets bitmap of button events.

Note
This is a non-blocking function.
Parameters
current-> Pointer to int, where current buttons state will be stored.
Note
If current buttons state is not wanted, current should be NULL (0).
Returns
If a certain button is in a different state comparing to the previous call to this function, the corresponding bit is returned as 1.
