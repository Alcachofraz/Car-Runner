◆ SPI_ConfigTransfer()

int32_t SPI_ConfigTransfer ( int  frequency,
int  bitData,
int  mode 
)

Configure transfer settings.

Parameters
frequency-> Transfer SPI rate.
bitData-> Bits per transfer. Must be a value between 8 and 16.
mode-> CPHA and CPOL information. Only the 2 lsb's will be read. Should be [0, 3].
Returns
0 if successfully configured, or -1 if bitData is out of range.
Note
This function must be called before 'SPI_Transfer()'.
