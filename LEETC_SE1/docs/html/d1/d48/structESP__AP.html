◆ bgn

char ESP_AP::bgn[2]

Consists of 3 bits:

  • Bit0 is for 802.11b mode;
  • Bit1 is for 802.11g mode;
  • Bit2 is for 802.11n mode.
    If the value of the bit is 1, the corresponding 802.11 mode is enabled.
