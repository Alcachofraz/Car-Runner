◆ WAIT_Init()

int32_t WAIT_Init ( WAIT  mode)

Initialises the wait API, using the desired resource.

Parameters
mode-> Which resource to use (see enum WAIT).
Returns
0 if initialisation successed, -1 if fails.
Note
This function must be called prior to any other WAIT functions.
Only 1 type of resource may be initialised for each timer, i.e. if TIMi is initialised, trying to initialise IRQi will fail, and vice-versa.
