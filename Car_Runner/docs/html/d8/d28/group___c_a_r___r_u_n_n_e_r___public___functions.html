◆ idle()

void idle ( void  )

Idle Mode. The fundamental state of the program. Shows RTC time and best scores, alternately.

Note
Use B1 and B2 to scroll between 'Time' and 'Best Scores'.
Use B3 to enter Game Mode.
Hold B1 and B2 for a certain amount of time to enter Configuration Mode.
This function never clears the LCD display, upon exit.
This function uses Timer0 and Timer1.
When entering Game Mode, it's guaranteed both timers are no longer busy.
When entering Configuration Mode, it's guaranteed that Timer0 is no longer busy, but not Timer1 (since it won't be used).
